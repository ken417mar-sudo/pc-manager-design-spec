## 导出脚本
本仓库已包含主工程的切图导出脚本（`scripts/export-slices.mjs`），输出到本仓库 `assets/slices/`。

在本仓库根目录运行：

```bash
FIGMA_TOKEN=你的token node scripts/export-slices.mjs
```
