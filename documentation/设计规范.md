# 电脑管家桌面端 — 设计规范

**完整规范与切图清单请以根目录 [spec.md](../spec.md) 为准。** 本文件仅作入口说明。

- **Spec 约束**（新增/修改必守）：颜色、字体、圆角、间距、阴影、组件状态等一律使用 **`tokens.css`**（根目录）中的变量；Token 未覆盖时须用户确认后再补充。详见 [SPEC-CONSTRAINTS.md](./SPEC-CONSTRAINTS.md)。
- **Token 文件**：本仓库以根目录 **`tokens.css`** 为唯一来源；`tokens/design-tokens.css` 通过引用该文件保持兼容。
- **结构化数据**：`resources/layout.json`、`resources/icons.json`、`resources/component-states.json`。

开发接入时引入 `tokens.css`，按 `spec.md` 中的切图清单与组件尺寸实现。
