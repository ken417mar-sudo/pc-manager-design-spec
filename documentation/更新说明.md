# 更新说明

## 内容来源

本 spec 仓库内容**从主工程摘出并补充**：

- **主工程路径**：`Codex/管家测试`（或同仓库下的 `design-spec/`、`docs/`、`src/renderer/`、`scripts/`）。
- **设计规范**：与主工程 `docs/design-spec.md`、`design-spec/spec.md` 一致；本仓库以根目录 **`spec.md`** 为唯一规范正文，`documentation/设计规范.md` 为入口说明。
- **Token**：与主工程 `design-spec/tokens.css`、`src/renderer/styles/tokens.css` 一致；本仓库以根目录 **`tokens.css`** 为唯一 token 源，`tokens/design-tokens.css` 通过引用该文件保持兼容。
- **切图导出**：`scripts/export-slices.mjs` 来自主工程，输出目录已改为本仓库的 `assets/slices/`；切图清单与 Figma 节点 ID 见 `resources/icons.json`。
- **布局与组件**：`resources/layout.json` 的尺寸与位置来自主工程 `src/renderer/styles/app.css`；`resources/component-states.json` 为组件状态约定。

## 设计 spec 仓库结构说明

- **设计规范**：以根目录 **`spec.md`** 为准；`documentation/设计规范.md` 仅作入口并指向 spec.md。
- **Token**：以根目录 **`tokens.css`** 为准；`tokens/design-tokens.css` 引用该文件，开发推荐直接引用 `tokens.css`。
- **资源**：`resources/` 下为布局、图标、组件状态等结构化数据，与主工程及 Figma 保持一致。

## 版本与来源

本设计 spec 来自电脑管家桌面端主工程，与主工程中的 `design-spec/`、`docs/`、`scripts/` 及 Figma 变量/子图层保持一致。
